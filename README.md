# Лабораторная работа №9-10:Продвинутое ООП на Kotlin

## Описание
Лабораторная работа посвящена изучение продвинутых концепций обьектно-ориентированного программирования в Kotlin.

## Структура проекта
Проект содержит примеры реализации продвинутых механизмов ООП, а так же пояснения к ключевым концепциям.

## Как запустить проект
1. Клонируйте репозиторий:
```bash
git clone <URL репозитория>
```
2. Откройте проект в Intelloj IDEA.
3. Запустите любой пример через контекстное меню или напрямую из 'main'.
